# Security Policy

## Supported Versions

We only provide security updates for the latest major release.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover any security related issues, please report them via the GitHub issue tracker or contact the maintainers directly. DO NOT disclose security vulnerabilities publicly until they have been addressed.
